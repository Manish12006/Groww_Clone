# Groww Clone – Full Stack Investment Dashboard (MERN Stack)

A full stack stock investment and portfolio management platform inspired by Groww. Built using the MERN stack with Python integration, this project focuses on real-time stock analysis, secure authentication, and an intuitive user experience for simulated trading and portfolio tracking.

## Project Duration
November 2024 – December 2024  
Associated with University of Mumbai

---

## Project Overview

The Groww Clone is a fully functional investment dashboard that allows users to simulate stock trading, manage portfolios, and analyze market data through interactive visualizations. The application integrates a Python Flask server for real-time stock analytics using Plotly, demonstrating seamless communication between JavaScript and Python-based backends.

---

## Key Features

- User authentication with email-based OTP verification for signup and login
- Simulated stock buying and selling with virtual fund deposits
- Real-time stock market data fetched using the yFinance API
- Interactive stock graphs generated using Python Flask and Plotly
- Dynamic dashboard displaying portfolio value and holdings
- Complete transaction history for user review
- Appointment booking system with expert details via interactive forms
- Fully responsive user interface with modern design standards
- Backend powered by Node.js and Express with MongoDB for data storage
- API-based integration between MERN frontend and Python analytics server

---

## Tech Stack

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB
- Analytics Service: Python, Flask
- Data Visualization: Plotly
- Market Data API: yFinance
- Styling: HTML, CSS, Bootstrap
- APIs: REST
